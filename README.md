# Blå Tur - Årlig Reise

En nettside for en gruppe på 8 gutter som reiser sammen hvert år.

## Teknologi

- **Next.js 14** - React framework med App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **ESLint** - Code linting

## Komme i gang

Installer avhengigheter:

```bash
npm install
```

Kjør utviklingsserveren:

```bash
npm run dev
```

Åpne [http://localhost:3000](http://localhost:3000) i nettleseren for å se resultatet.

## Struktur

- `/app` - Next.js App Router pages og layouts
- `/app/page.tsx` - Hjemmeside
- `/app/trips` - Turside
- `/app/members` - Medlemmsside

## Bygge for produksjon

```bash
npm run build
npm start
```
